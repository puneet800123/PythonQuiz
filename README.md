## Code Your Own Quiz
### About : 

User is asked to input a number as per the desired difficulty level

1- Easy Difficulty
2- Medium Difficulty 
3- Hard Difficulty

Then Question is displayed as per the Difficulty level and if answered correctly a congratulations message is displayed at the end.
